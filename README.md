# 2026 夏季中国行程

这是一个用于展示 2026 夏季中国旅行安排的静态网站。

## 行程

- 日期：2026 年 7 月 31 日至 8 月 4 日
- 路线：北京南 → 徽州古城 → 呈坎 → 黄山 → 宏村 → 西递 → 屯溪 → 南昌
- 形式：单页静态网站，包含日期切换、时间线、住宿提醒、预算快照和出行准备清单

## 本地查看

可以直接用浏览器打开 `index.html`，也可以用任意静态服务器启动这个文件夹。

```powershell
python -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 临近出发前需要复核

- 高铁时间和余票
- 景区门票价格
- 黄山索道、峰顶和西海大峡谷开放情况
- 汤口上山前行李寄存点、收费和取件时间
- 呈坎和宏村鱼灯巡游排期
- 西递门票、接驳交通和当日可用游览时间
- 黄山天气、日出和日落时间
- 最新天气预报；页面中的去年同日气温来自 Open-Meteo 历史天气数据，仅作参考
